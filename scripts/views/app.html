jQuery Plugin - Basic Toggle Switch

Download Code & Styles

https://github.com/vail130/Basic-Toggle-Switch

Standard Initialization

<span class='switch'></span>
var $switch = $('.switch').basicToggleSwitch();

Active (data-state='on')

<span class='switch' data-state='on'></span>
var $switch = $('.switch').basicToggleSwitch();

Referencing the Plugin

Save a reference to the element that the initializing method returns for later use. The original HTML will always be replaced by the same switch template, so selecting by attributes from the original HTML will not work.

State

Initializing the plugin on a DOM element returns the newly created jQuery selection of just the one object, on which you can call the plugin method with 'state' as the first argument to get a string representation of the switch's state: 'on' or 'off'.

var state = $switch.basicToggleSwitch('state');
['on', 'off'].indexOf(state) > -1 // true

You can call the plugin method with 'active' as the first argument to get a boolean representation of the switch's state: true or false.

var active = $switch.basicToggleSwitch('active');
[true, false].indexOf(active) > -1 // true

Events

var $switch = $('.switch').basicToggleSwitch({
    on: function() {
      $('.switch-output').html("<%- "Switch is on" %>");
    },
    off: function() {
      $('.switch-output').html("<%- "Switch is on" %>");
    },
    toggle: function() {
      $('.switch-output').append("<%- "Switch is toggled." %>");
    }
  });

Multiple Switches on a Page

To create multiple switches on a page, only call basicToggleSwitch() on selections with a single element. Use jQuery's each() iterator to create multiple switches with one selection.

<span class='switch' data-state='on'></span>
<span class='switch' data-state='on'></span>
<span class='switch'></span>
var switchArray = [];
$('.switch').each(function() {
  switchArray.push($(this).basicToggleSwitch());
});
